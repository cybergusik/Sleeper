# Sleeper App
A minimalistic native utility for disabling automatic sleep mode when closing the lid of a MacBook. After launch, it is located in the tray. There is an option for autostart
## Usage

**clone this repo:**
  ```bash
  git clone https://github.com/cybergusik/Sleeper
  cd Sleeper
  ```

**run build script(xcode must be installed):**
  ```bash
  bash ./build.sh
  ```
